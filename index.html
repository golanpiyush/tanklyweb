2. Data We Collect §2

Account Data (Optional — Cloud Sync Only)

If you enable cloud backup, we collect:

  • Email address (for authentication via Supabase Auth)
  • Encrypted password hash (we never see your plaintext password)
  • User UUID assigned at signup

App Content (Your Tank Data)

Data you actively create inside the app, stored in our Supabase database:

  • Tank names, volumes, types, setup dates
  • Water parameter readings (pH, ammonia, nitrate, temperature, etc.)
  • Livestock entries and maintenance logs
  • Photos you optionally add to tanks
  • Expense records and equipment logs (Pro)

3. How We Use Your Data §3

We use the data we collect exclusively to:

  • Provide the App: Sync your tank data across devices, send maintenance reminders
  • Authenticate you: Verify login, manage Pro subscription status
  • Improve the App: Analyse anonymous crash/usage patterns to fix bugs
  • Legal compliance: Record your acceptance of Terms and Privacy Policy (stored in user_legal_acceptance table with RLS — only you can read your own row)
We do not: sell your data, use it for advertising, share it with third-party marketers, or run profiling algorithms on your aquarium habits.
🛡️
4. Data Storage & Security §4

Infrastructure

User data is stored on Supabase (PostgreSQL on AWS infrastructure) with the following protections:

  • Row-Level Security (RLS) — every table has policies so users can only access their own rows
  • HTTPS / TLS for all data in transit
  • AES-256 encryption at rest
  • Supabase Auth handles password hashing (bcrypt)

Free Tier — Local Only

If you use Tankly without creating an account, all data stays on your device only. Nothing is transmitted to our servers.

Data Retention

  • Your tank data: retained as long as your account is active
  • Crash logs: anonymised, deleted after 90 days
  • After account deletion: all personal data removed within 30 days
